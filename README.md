# Songbook App

Offline-first songbook built with SvelteKit.

## Development

```bash
npm install
npm run dev
```

If you see a TypeScript error about a missing `.svelte-kit/tsconfig.json`, run:

```bash
npm run prepare
```

## Build

```bash
npm run build
```

This project uses Tailwind CSS, Supabase, three.js, Framer Motion and PWA support.
